move
Place the mouse cursor on any visible location inside the game screen and then click left mouse button to make the character to move to such location.
“Auto Path Finding” function is supported so the character will be able to move to the desired location through the complicated paths and obstacles automatically.

If return card is possessed, moving to another place can be done really fast, specific usages are as below.

-School Return Card: Moves character to the character’s school main center.
-Start Point Card: Moves character to the character’s starting location.
-Back Point Card: Moves character to the last location where a return card had been used or dead location.
-Buddy Teleport Card: Moves character to the friend’s current location.
bus
With Bus Pass possessed, it will be possible to ride the buses that are located in many places and moving to a location in a different map also can be done at once.
Teleport(connect) card
If return card is possessed, moving to another place can be done really fast, specific usages are as below.

-School Return Card: Moves character to the character’s school main center.
-Start Point Card: Moves character to the character’s starting location.
-Back Point Card: Moves character to the last location where a return card had been used or dead location.
-Buddy Teleport Card: Moves character to the friend’s current location.
battle
Players can attack monsters which are not a party member nor NPC and there are two types of attack,
'General Attack' which can be initiated with click of left mouse button and
‘Skill Attack’ which can be initiated with click of right mouse button after selecting a skill to cast.
Combat/Non-combat Zone
All the combat action is forbidden even holding a weapon in a non combat zone
How to autoattack
Place the cursor on the target to attack and press and hold ‘Ctrl’ key and click left mouse button or
press ‘ESC’ to view the ‘Option’ menu and check the box next to the ‘Auto Attack’ to attack a target with just one click of mouse button.
Skill
This is a term used to indicate special attacks or defense techniques and skills can only be used when first learned the skill using ‘Skill Scroll’.
In order to learn a skill, requirements must be met for each skill.
How to learn a skill
First acquire ‘Skill Scroll’ through a quest or from a NPC.
Place the cursor on a ‘Skill Scroll’ located in the inventory to see the requirements to learn the skill.
(Unmet requirements will be displayed in red)
When all the requirements are met to learn a skill, place the sursor on the ‘skill scroll’ and click it with right mouse button to learn it.
But, once learned skill cannot be cancelled so please be careful.
Skill level up
All skills can be upgraded up to maximum 9 levels
'Skill window' is activated by using the shortcut key (K). The requirements for next level is displayed when the cursor is over target skill.
If all requirements are satisfied (displayed in green text), the skill can be upgraded by clicking '▲' button in right side
(If the skill points is not enough, the upgrade cannot be done)
But, be careful with upgrade You cannot undo the applied skill points.
how to use skill
Press ‘K’ to open the Skill Window.

1. Drag and place the desired skill to use (Most used skill) into the skill slot located on the top left corner of the game screen.
(In case left skill slot is not displayed, press ’`’ button.)

2. Select the desired skill to use by using ‘Shortcut Key 1~0’ buttons or clicking it with left mouse button.

3. Place the cursor on the target to attack and press the right mouse button to cast the skill.

Use F1~F4 buttons to convert the pages of skill slot.
How to use item
Items acquired through trades or battles can be used as their uses.
Item Grade
- General Item
Indicate items that have the basic values

- Rare Item
Indicate upgraded items that contains the extra or special features

- Unique Item
Indicate special items that provide more features than Rare Item. Hard to find items

- Quest Item
Indicate items that can be used only for quest and cannot be used otherwise
Item Type
- Consumable Item
Items which can be easily purchased in shops and consumed as they are used. Automatically saved in the inventory as acquired.

1. Open the Inventory window using 'I' button.

2. Drag and place the frequently used items into 'Item Slot' loacted in the top of the game screen with left mouse button

3. Assigned items in 'Item Slot' can be used with 'Q,W,E,A,S,D' buttons

- Non-consumable item
Items which are not destroyed or consumed such as weapons and armors and cannot be assigned in 'Item Slot'.
Item Equip
Acquire Equipment (Equip-able Item)

Open Inventory window using ‘I’ Button.

1. When the slot for the desired equipment is empty:
Drag and place desired equipment with left mouse button to a proper slot and click left mouse button again to equip.

2. When the equip-able slot for desired equipment is not empty.
Drag and place desired equipment with left mouse button to a proper slot and click left mouse button again to equip then the currently equipped item will be selected,
just move it to the empty slot in the inventory window and click left mouse button again to put it down.
Revision
Most of the weapons and armors can be revised.
If the revision is succeful, attack and defense value will be upgraded to help the game but
if it fails then currently acheived revision values or item itself might can vanish.
How to Revise
If the item desired to revise iis equipped, it should be unequipped and placed in the inventory. (Revision cannot be done when item is equipped.)

1. Select the revision item (Grind Stone, Coating Oil, and etc) by clicking it with left mouse button.

2. Place the cursor on the target item to be revised and click the right mouse button to revise it.

3. Revision result will be displayed in chat window. If successful, the new value can be checked by placing the cursor on the item.
NPC
Unlike the mosters that attack players, NPC provide information and quests and play locker or store roles.
Locker
Approach the locker NPC and start the convensation to see the 'Use Locker' menu.
Select 'Use Locker' menu to see the store the items in the inventory to locker to keep more items.

1. Personal Locker
If 'Locker Expansion' is used, the double or more spaces of Locker can be used.

2. Club Locker
As the Club rank gets higher, more space will be provided.
Store
Approach the Store NPC and start the conversation to see the 'Trade Item' menu.
Select 'Trade Item' to start the trade and see the store product window and character's inventory window.

Buying - Select desired item in 'Store' window with left mouse button.

1. If enough gold for the item is possessed, then the selected item will be saved in the inventory and the gold will be paid to store.

2. For consumable items, the quantity input window will be displayed.

3.Once the quantity of item is entered and the gold is enough, then same amount of items will be saved in inventory

Selling - Select a item to sell in the inventory with left mouse button and move it to Store window to sell at the automatically calculated price.
Personal Store
What is Personal Store System?
- Personal Store is a game system which let players to make trades regardless of their locations.
- As a student, players must acquire permit from the school to open a Personal Store.

How to use?
- ‘Personal Store Permit’ is needed to open a store.
- ‘Personal Store Permit’ can be purchased from merchant.
- Open the store menu by clicking the permit in the inventory with right mouse button.
- Set the name of the store. (ENG maximum 40 letters)
- Move the items for sale as when buying items and input the prices. Maximum 24 items can be registered for sale.
- Press 'Open' button to start the store. Character which opened the store cannot move until the store is closed. (But, once the store is opened, it is not possible to change the contents.)
- Personal Store Permit purchased from merchant has limit for number of times to open the store but there is no limit how long a store can be stay opened.
- Once the store is opened, other players will be able to see the store name and they will be able to shop the items they want at a desired price from a store, just like buying items from the merchant.
Setting Starting Point
When this option is displayed, players can set such location to be their starting point when the character became unconscious during combats.
Duel
System prepared to let players to compete with each other. Possible to be progressed under mutual consent.
How to duel
How to request duel?
Place the cursor on a desired character to duel against.

1. Click with left mouse button while holding down the ‘Shift’ button to view the menu arranged with 5~6 Icons.

2. Select 4th icon from the left to activate the ‘Duel Setting Menu’.

3. Set the duel regulations and press ‘Confirm’ button.

4. Duel will begin once the opponent player presses ‘confirm’ on the duel menu.

5. Duel starts as the ‘Fight’ message appears on the screen.
Duel Regulations
- Use of Potion
: Set total number of potions which can be used during the battle, Default value is ‘0’.

- HP Reinforcement
: Just before the battle, HP of each character can be adjusted at certain rate and it can be set differently.

- Area Limitation
: If Area Limitation is set then there will a ‘Ring’ set around the contestants and a player who moves out of this area will be defeated.
How to play duel?
- Once going into the battle stage under mutual consent, the fight will be continued for 180 seconds or until HP of one side becomes 0 or lower. During the battle all the other Duel requests will automatically rejected.

- There will be no numeric decrease whether win or lose the duel.
Interface
There are several menus place on the game screen to play the game. These menus are movable and its window sizes can be adjusted for some menus.
Character Window
Press ‘C’ button to view the character information. For more details, please see the Ability Description.
Party Window
Press ‘P’ button to view the window. If a party is formed then party members’ information can be checked. What buffs do the party members received, where they are located and etc.
Inventory Window
Press ‘I’ button to view the window. Consumable items, armors or weapons can be equipped or un-equipped and stored in this window. Item revision must be done in this window also.
Skill Window
Press ‘K’ to view the window. Skills and their conditions which a character can use can be confirmed in this window and skill level up can be done as well.
Chatting Window
As logging into the game, this window is activated in bottom left corner of the screen and it is used to communicate between the players. It can be used conveniently with the ‘Enter’ button.

- All : Shows all the message there can be in a game such as Chatting, Announcement, and etc.
It is possible to set detailed options to select ‘General Chatting’, ‘Party Chatting’, ‘Club Chatting’, ’Alliance Chatting’ and ‘Channel Chatting’ when ‘All’ button is pressed.

- General : Shows general chatting only. Input ‘/say’, ‘/s’ space and message to use the general chatting.

- Whisper : Shows the whisper messages only. To use the whisper chatting use ‘Friend List’ or input ‘/whisper target character name’, or ‘/w target character name’, ‘@ target character name’ space and message. Clicking other characters name shown in the chatting window will automatically lets player can send the whispering message to that character. Last, if player wants to send a whisper to the same character whispered with then just input ‘/r’ and press spacebar to whisper to that character again.

- Party : Shows the party chatting content only.
Input ‘/party’, ‘/p’ or ‘#’, space and message to chat only with party members.

- Club : Shows the club chatting content only.
Input ‘/club’, ‘/c’ or ‘%’, space and message to chat only with club members.

- Alliance : Shows the alliance chatting content only.
Input ‘/union’, ‘/u’ or ‘!’, space and message to chat with the members from alliance.

- Channel : Shows the ‘Region Channel’ messages and ‘Party Search Channel’ messages which allows players to search party only.
Input ‘/1’, space and message for Regional Chatting.
Input ‘/2’, space and message for Party Search Chatting.

- System : Shows only game system message and announcements.

- If ‘Megaphone’ is in the inventory then input ‘/yell’, ‘/y’ or ‘$’, space and message to shout out to all the players in the same server.
Trading Window
Automatic pop up window for making trade with other characters, present the item and gold for trade and when both agree, trade will be completed. For the fair trade, please make sure to check the item information and amount of gold before approving the trade.
Resurrection Window
During Battle or hunting, a character might die from various causes.
This is a window activated at a moment like this to decide the resurrection of the character.
Character will be resurrected at the appointed starting location and remember that certain amount of EXP will be lost from the character level 10 or higher.
Using ‘Resurrection Rosary’ will let the player to be resurrected at the vary spot of death without any EXP loss. But, if player is resurrected with the help of ‘Resurrection’ skill, EXP will be lost.
Mini Map
Press 'M' button to view the window.
This will not only show the current location of the player but also the NPCs’. Plus, it show the location of the party members if joined in one. Size and location of this window can be modified as preferred.
Option
For a better game playing environment, this lets players to adjust various functions of the game according to their PC specifications.
- Display : Adjust screen size and maximum colors.
- Shadow : Set character shadow.
- Detail : Set character display grade
- Distance : Set how much background to show.
- Effects : Set to show buff skill effects or not.
- BG Shadow : Set to show BG shadow or not.

Press ‘ESC’ key to view this window to end the game session while playing the game.

Ability Description
Various information are saved in a player created character and certain values of these saved information are changed to level up a character.
Name
This is character’s original name. Name cannot be changed once set.
System forbid to use the words that are swearing to and abusing others as a character name.
Department
This indicates the class which character belongs and currently Brawler, Swordsman, Archer, Shaman, Extreme and Science are prepared as default classes.

-Brawler : Melee type, fast attack speed and high evasion rate allows chain attacks.
-Swordsman : Close range combat type, speed is a little slow but has strong defense.
-Archer : Long range combat type, strong attack power and Long range attacks.
-Shaman : Hybrid combat type (Close range+Long range), assist other players and various attack skills.
-Extreme : Hybrid combat type (Close range+Long range), fast attack speed, strong attack power and long range attacks.
-Science : Long rage assist type, a little weak power but various special effects.
Level
Level can go up as the EXP accrues and the 3 stat points and 1 skill point rewarded at the level up allow users to raise a character with unique characteristics.
Power
This has effects on ‘Melee' and 'Shooting’ attack power. This is a very important value to be considered for the classes fight in close range such as Brawler and Swordsman.
Vitality
This has effect on maximum 'HP' value.
intelligence
This has effects on 'Magic Attack' and 'MP'.
This is a very important value for shaman which uses magic power to attack.
Dexterity
This has effects on various elements such as 'Defense', 'Shooting', 'Magic Attack’ and 'Melee'. Effectiveness is different depending on the class of a character.
Stamina
Influence on SP
If Stamina is low, the attack rate and power can be decreased by maximum of 50%.
HP
This is vital force and when it becomes ‘0’ then character dies.
It can be recovered by consuming food or medicines but it also recovers itself very little in every seconds. Equipping recovery rosary item can help increase its recover speed.
MP
This is amount of mental power which needed to cast skills.
It can be recovered by consuming food or medicines but it also recovers itself very little in every seconds. Equipping certain item can help increase its recover speed.
SP
This is the amount of Stamina and when it is drained accuracy rate and attack will be decreased 50% in the battle. It can be recovered by consuming food or medicines but it also recovers itself very little in every seconds. Equipping certain item can help increase its recover speed.
EXP
This is an acquired experience points through battles.
Certain accrued amount of EXP will level up the character.

In following cases EXP will be acquired.
- Eliminating monster with a name in red color.
- Completing a quest which reward EXP as a prize
- When EXP get effect has been acquired from Question Box.

EXP can be lost in following case.
- When a character of level 10 or higher died
Fight level
This is a value of close range combat ability which is mostly effected by 'POW' and 'Dex'.
Shooting Power
This is affected by 'Dex' and 'Pow' and effects the shooting attack ability.
Attack Power
This is affected by character basic stats and equipped weapon and indicates the actual damage which can be done on the target.
Defense Power
This is affected by character basic stats and equipped armors and indicates the blocking value when attacked.
Resistance
This indicates the resistance against magic attack with attributes such as fire, ice, electric, poison and none. If the resistance value is high, defense against such attribute will be strong and it will also have the resistance against abnormal conditions as well.
Class Info
Default 6 classes have different abilities and growth rate is different and therefore every class has different way to proceed.
Brawling
Melee type which uses arms, legs, head and all other body parts to attack fast in series with plenty of stamina.
As the growth type, character can be raised as ‘Fist’, 'Kick’ or ‘Energy’ fighting type.
Swordsmanship
Close range combat type using swords and blades with great HP and defense ability.
As the growth type, character can be raised as ‘Sword’, 'Blade’ or ‘Energy’ type.
Archery
Long range combat type using bow with great accuracy rate and attack power.
As the growth type, character can be raised as ‘Fast bow’, 'Strong Bow’ or ‘Energy bow’ type.
Shamanism
Uses magic in combat. As the growth type, character can be raised as close, medium and long range combat type.
Extreme
Hybrid combat type which uses every weapons without Science's.
As the growth type, character can be raised as ‘Melee’, 'Archery’ or ‘Magic’ fighting type.
And Extreme can switch to use two types of weapon.
Science
Long range combat type which uses guns with various effects.
As the growth type, character can be raised as ‘Debuff’, 'Restriction’ or ‘Summon’ fighting type.
Party
Forming a party will provide extra EXP and easier game play. The EXP distribute standard will be set by the lowest level character in a party and the distribution percentage will be adjusted. But, distributed EXP percentage would decrease as level difference with the lowest level is greater.
How to form a party
Party Master places the cursor on a character to recruit into the party.

1. Display the menu with 5~6 icons by clicking the character with left mouse button while holding the ‘Shift’ button.

2. Click the third icon from left to send ‘Party Invitation’ message.

3. Character will be joined in the party once accepting the invitation. But, only when such character is not in another party.
About party progress
'- 2 to maximum 8 members can form a single party and the members will be sharing each other’s information.

'- If members are in certain distance, acquired EXP by a member in a combat will be distributed to the other members.

'- If a member has moved to a different map, character name will still be in the party list but the gray color will be displayed and the information will not be refreshed.

'- Click withdraw button on the party menu (‘P’) to withdraw from the party.

'- Party master can expel a member or disband the party using the party menu (‘P’).
Disbanding Party
- Withdraw
: when a member decides to withdraw him or herself voluntarily.

- Expel
: when a member is forced out from the party by the party master’s rights.

- Log-Out
: When logged out from the game due to a PC error or member’s own intention to do so.

- When the party master withdraws or just one member is left then the party will be disbanded.

Club
Maximum 60 members can be joined in a club and players will be able to participate in the Lead Club Battle.
How to create a Club
-Form a party of 5 or more
: Such party’s master should apply for the registration to Student Director in each school Admin Office. ('Ask about club'->'Create Club')

-Newly created club can hold 15 members at the start and as the club rank gets higher maximum 60 members can be recruited.

*Club Creation Conditions
- When party master’s life point is 0 or higher.
- When party master is level 20 or higher.
- When all the party members are not joined in any other clubs.
- When the applicant can afford the fee
Changing Club Mark
Default club mark will be provided at the registration but club master can register or modify his or her own club mark. Mark must be produced size 16 x 11 in 24bit BMP or 32bit TGA format. When inputting the club mark file name, file extension must be entered together.
Club Chat
Input % and message to send the message only to the club members, click a club member’s name in the club member list to send the whisper.
Joining Club
This is a right given to the Club Master only.
Allows Master to send a message to a character to join its club and when target character accepts the invitation he or she will be recruited.
Dismiss / Withdraw
Club members can withdraw and Club Master can expel certain club member.
Expelled or withdrawn character cannot join another club for one day.
But, it is possible to create a new club.
Disband
Only club master has the right to disband the club and waiting period will take 7 days to be completed.
It is possible to cancel the disbandment during this waiting period.
Club Master must be aware of the EXP loss which amount would be same as dying once.
Alliance
Maximum 3 clubs can enter into an alliance.
‘Alliance’ related menu will be activated in the club menu and it will show you allies.
Separate Chatting channel exists for the alliance members.
During the Lead Club Battle, it is not possible to attack between the members of allied clubs unless on purpose.
Character which can authenticate during the Lead Club Battle is limited to a one person among the allies.
In order to form an alliance, a club master should click ‘Alliance Request’ icon in the menu viewed by ‘shift’ + clicking on the other club’s master and alliance will be formed upon mutual agreement.

Club rank-up can be processed by talking to a Student Director at Admin Office in each school main center.
But, club rank cannot be degraded once it has ranked up.

Lead Club Battle
Since the establishment of the 4 schools, Saint Foundation have focused all the automated financial and management task functions of each area into their local Data Processing Room.
As the phenomenon outbursts, all the data processing rooms have stopped operating but students devoted recovery works have restored some of its system little by little.
Therefore, the tax (Trading Fee) which has not been collected for some time started to be withdrawn automatically from everyone’s account and this was applied to the 3 school areas and SG, MP, PH and Up towns.
Data Processing Room is supposed to be managed by a person who was given special rights but teachers of 3 schools have decided to handover this right to the students to vitalize their learning and build the sense of belonging. That is how the Lead Club system was first established.
How to Play Lead Club Battle
Day & Time : Every Sunday from 20:00, for 1 hour.
Location : Each Regional Data Processing Dept. - Only bus can be used to get there for security reasons.
Limitations : It is not possible to be a lead club in more than one region at the same time.

-Announcement will be made 1hour, 30minutes and 10 minutes prior to the Battle.
-Lead club members can enter their region's data processing room freely until the rights are still standing.
-At 20:00 hours, the battle begins in each region's data processing room with an announcement.
-At the same time in the regional data processing room, unlimited PK mode will be started between different club members
-If the attacking side can authenticate for one minute at the authentication terminal then the attacking side will be switched to defend and all the other club members will be forced out to the entrance of data processing room.
-Authentication can only be done by the Club Master who has the Authentication CD.
-Last minute defending club will become the lead club when the time is up.
-No EXP penalty will be applied when killed during the Lead Club Battle.

->If the battle ends in a draw then the commercial rights will be handed over to the Monsters and all the players will be charged of 5% trading fee.
Managing Trading Fee
-Club which acquired the control of region’s data processing room will maintain its right until the next Lead Club Battle ends.
-Club Master of a club which has won the control of one of the four regions must set the trading fee rate between 0~50%. (Default 5%)
-Collected trading fees of each region will be remitted to the lead club of such region lock in every one hour.
Regional Data Processing Room
This place can only be reached by bus. It is possible to change starting point at the entrance but it is not possible to change it inside the Data Processing Room.
Authentication CD
This can be purchased at each data processing room.
But, it has short durability due to used self made material instead of usual CD material. (Duration 1 Day)
